 📊 Spotify Data Dashboard
📌 Project Overview

This project is a Spotify Data Dashboard built using Power BI to analyze music trends, artist performance, and song insights.
It provides interactive visualizations to explore key metrics across different dimensions.

🚀 Features
🎧 Home Page – Overview of key KPIs and highlights
📊 Overview Page – General trends and performance analysis
🎤 Artist Page – Artist-wise insights and comparisons
🎵 Song Page – Song-level analysis and metrics
